# TEs Atlas and Impacts on Plant Evolution
This repository contains the code for the manuscript titled "Comprehensive Analysis of Transposable Elements Across 352 Plant Species Reveals Evolutionary Patterns"

# Content

Pipelines and relevant scripts for:

- 01.Genome_Annotation
- 02.Function_Gene_and_ncRNA Annotation
- 03.TE-associated_Genes_and_ncRNAs_Detection
- 04.TE_induced_Gene_Duplications_Analysis
- 05.Assessing_the_Impact_of_TEs_on_Genetic_Functions
- 06.Lineage_Specific_Feature_Selection
- 07.Phylogenetic_Analysis
